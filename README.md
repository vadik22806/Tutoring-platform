# StudyEasy - Платформа для поиска репетиторов

## Описание
Веб-платформа для поиска репетиторов и организации онлайн-занятий.

## Функционал
- ✅ Регистрация и авторизация (email/телефон)
- ✅ Два типа пользователей: ученики и репетиторы
- ✅ Создание и управление занятиями
- ✅ Избранное (сохранение занятий)
- ✅ История записей
- ✅ Валидация на лету с AJAX
- ✅ Selenium тесты (4 теста)

## Технологии
- Python 3.12
- Django 3.1
- MongoDB (djongo)
- Bootstrap 5
- Selenium
